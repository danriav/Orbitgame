# DarkOrbit 2.0 - Guía de Relación de Assets (Naves)
Este documento sirve como referencia para saber qué Modelos (Mesh) corresponden a qué Materiales/Texturas.

## Metodología de Construcción
Para naves que no tienen nombre explícito en el Mesh, se utiliza el índice de AssetRipper.
Regla general: `default_ID.asset` suele usar `M_DO_1_ID.mat` o un material específico `ship_nombre.mat`.

## Notas Técnicas para el SceneBuilder
Al asignar materiales por código, si la nave se ve **Blanca**, el script `MainSpaceSceneBuilder.cs` realiza lo siguiente:
1. Cambia el Shader a `Standard`.
2. Busca la textura en la propiedad `_Diffuse` y la mueve a `_MainTex`.
3. Busca la textura en la propiedad `_Normal` y la mueve a `_BumpMap`.
4. Activa el Keyword `_NORMALMAP`.
